## Accepted Risks Log

### T-08.2-SC -- typescript@6.0.3 devDependency (accepted)

- **Risk:** Supply-chain tampering via the pinned `typescript@6.0.3` devDependency used by the
  non-shipped compile harness (`extract-samples.mjs` -> `tsc --strict`).
- **Why accepted:** No new package installs occurred this phase. `typescript@6.0.3` was pinned and
  audited in Phase 7 and is unchanged. It is a devDependency of the NON-shipped `lz-refactor-workspace`
  harness only -- the shipped plugin (`plugins/lz-tdd/`) is dependency-free Markdown, so this
  dependency never reaches an installing user.
- **Verification:** `HEAD:.claude/skills/lz-refactor-workspace/package.json` `devDependencies` = `{ "typescript": "6.0.3" }` (single, pinned). The only package.json change this phase added a script step, not a dependency.

- **No-verbatim gate is a soft heuristic.** `check-hygiene.mjs` no-verbatim check is WARN-level only
  (flags double-quoted runs >= 120 chars); its own header (lines 7-8) states the authoritative
  no-verbatim gate is the D-07 oracle checkpoint + Phase-10 hygiene scan. This phase has NO owned
  oracle (D-06), so the assurance for T-08.2-IP rests on DST-04 authoring discipline + the
  code-reviewer's manual end-to-end trace (0 findings). The heuristic would not catch verbatim runs
  under 120 chars or unquoted verbatim; accepted as a documented limitation, not an open threat.
- **Hygiene gate scope.** `check-hygiene.mjs` walks only `SKILL.md` + `references/` recursively (the
  shipped skill tree), which is the correct scope for a "leak into committed public files" threat.
  The independent `git grep` over `plugins/` + workspace confirmed the shipped tree is clean.
- **Gate-hardening findings from 08.2-REVIEW.md (WR-01, WR-02, IN-01, IN-02)** are future-drift
  weaknesses in `check-functional.mjs` (e.g. C4 validates only the first `Functional alternative:`
  line). None affect the security dispositions above; all 55 current leaves are authored with exactly
  one line and the sibling checkers assert the catalog counts. Noted for Phase-9+ hardening.
