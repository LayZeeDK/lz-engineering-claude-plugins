| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|

Main context uses `ls` for names only -- NEVER Read/cat/git grep of `.oracle/` contents. |
| Fabricating catalog content instead of the oracle loop | Authoring a leaf from memory and committing it without gating it against the source -- plausible-but-unfaithful mechanics/examples. | blocking | Mandatory loop: author blind -> gate via `oracle-reviewer` (per-draft pass/revise/blocked/error) -> revise blind -> converge (all pass, or owner-accepted blocked). No leaf committed until it converges. |
| Spawning `gsd-executor` to run a content plan | The standard execute-phase spawns gsd-executor subagents in worktrees. That FAILS here: gsd-executor has no Agent tool (cannot spawn oracle/oracle-reviewer) and no AskUserQuestion (cannot do owner escalation), so it cannot run the clean-room loop. | blocking | The ORCHESTRATOR (main context) drives each content plan INLINE -- authoring (Write), gating (Agent -> oracle-reviewer), escalating (AskUserQuestion). branching_strategy=none; no worktrees (oracle agents are read-only). |
| Wave-3 chapter plan touches README.md or checker .mjs | A chapter plan editing shared files breaks file-disjointness and mis-reads RED-by-design checks as failures. | advisory | Wave-3 plans (07-05..07-09) author ONLY their chapter's leaf files. 07-10 fills README Ch.7-12 rows + runs the full battery. check-catalog mirror + check-crossrefs are RED-by-design until 07-10 and are NOT wave-3 gates. |
| Gating Ch.9/10/11 without a tight per-leaf SCOPE | Those source chapters contain dagger-marked (print-absent) OUT-OF-SCOPE entries; a loose SCOPE makes the reviewer flag them as dropped (`source-only`) items. | advisory | Pass SCOPE = the single refactoring each leaf covers; out-of-scope source items are then not DROPs. |
| Reusing the source's example domains/identifiers | Copying Fowler's example domain/identifiers/structure trips the `too_close_to_source` gate. | advisory | Author every `## Example` in a DISTINCT domain with original identifiers. |

**Do not proceed until these are understood.** The three `blocking` rows gate resumption.
