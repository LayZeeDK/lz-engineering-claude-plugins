### `references/functional-catalog/<slug>.md` (x19) (shipped content leaf, Before->After transform)

**Analog:** `plugins/lz-tdd/skills/lz-refactor/references/kerievsky-catalog/replace-conditional-logic-with-strategy.md` (header + body skeleton to mirror) and `move-accumulation-to-visitor.md` (the Direction:Away metadata cross-link line to model `Correspondence:` on).

The functional leaf keeps the kerievsky leaf's SHAPE (H1, then a metadata header block of `Label: value` lines, then `## ` body sections) but swaps the label names and section names per D-01/S13.2.

**Kerievsky header shape being mirrored** (`replace-conditional-logic-with-strategy.md` lines 1-8):
```
# Replace Conditional Logic with Strategy

Use when: a conditional selects between whole algorithms, and you want to vary or extend those algorithms independently of the code that runs them.

Direction: To/Towards
GoF pattern: [Strategy](../gof-catalog/strategy.md#strategy)
Composed Fowler primitive(s): [Replace Conditional with Polymorphism](../fowler-catalog/replace-conditional-with-polymorphism.md#replace-conditional-with-polymorphism), ...
```

**Functional leaf header maps to** (D-01, RESEARCH.md S13.2 template, lines 470-490):
```
# <idiom name>

Use when: <one-line smell/trigger; mirrored VERBATIM in the README index row>
Correspondence: dissolves-from -> [Strategy](../gof-catalog/strategy.md#strategy), [Bridge](../gof-catalog/bridge.md#bridge)
Keep the OO form when: <identity | coordinated mutable state | measured hot path | new-variant churn (expression problem) | house style>
```
- `Use when:` -- keep verbatim from the kerievsky label (same regex `^Use when:\s*\S`, so the selector-mirror check reuses the kerievsky idiom).
- `Correspondence:` REPLACES `Direction:` + `GoF pattern:`. Enum is EXACTLY `dissolves-from` or `alternative-to` (RESEARCH.md C2). The link(s) after `->` point at `../{gof,kerievsky,extra-patterns,fowler}-catalog/<slug>.md#<anchor>` and are the SAME link-string form as the kerievsky `GoF pattern:` line (line 6 above) and the `Composed Fowler primitive(s):` line (line 7).
- `Keep the OO form when:` REPLACES `Composed Fowler primitive(s):`. Header line (before `## Idiom`), NOT a body section (Pitfall 4).

**Body section mapping** (kerievsky `## Motivation`/`## Mechanics`/`## Example` -> functional `## Idiom`/`## Example`/`## When each fits`):
- `## Idiom` -- 1-2 lines, ORIGINAL wording. NO `## Mechanics` section (D-01 forbids it; Anti-Patterns list, RESEARCH.md lines 493-495). PARAPHRASE -- do not lift the research artifact's Norvig/Wlaschin quotes (Pitfall 1 / MEMORY DST-04 trap).
- `## Example` -- Before (OO) -> After (idiom), same two-fence "Before -- ... / After -- ..." layout as the kerievsky analog (lines 32-69). Mechanics live as ONE cross-link line inside `## Example`: `Same behavior; mechanics: [<Fowler/Kerievsky refactoring>](../<catalog>/<slug>.md#<anchor>), run tests between steps.` Every `ts` fence must sit INSIDE `## Example` (the extractor + C1/C6 assert this).
- `## When each fits` -- for N:1 leaves (1,2,3,4,8,9), one `### <Served pattern>` sub-heading per served pattern + EXACTLY ONE content line each (C5 cap). `### Visitor` -> anchor `visitor` via `githubSlug`, which the README row targets. 1:1 leaves get a single un-split paragraph, no `###`.

**Direction:Away metadata line -- the exact one-line cross-link precedent** (`move-accumulation-to-visitor.md` lines 5-6):
```
Direction: Away
GoF pattern: [Visitor](../gof-catalog/visitor.md#visitor)
```
This is the format the `Correspondence:` link mirrors: a bare `[Name](../<catalog>/<slug>.md#<anchor>)` on one metadata line. Same mechanism D-09 reuses for the reverse `Functional alternative:` links.

The 19 slugs + Correspondence enum values + served-pattern sets are the CANONICAL inventory in RESEARCH.md lines 138-165 (13 dissolves-from + 6 alternative-to). The checker's `LEAVES` constant encodes them.
