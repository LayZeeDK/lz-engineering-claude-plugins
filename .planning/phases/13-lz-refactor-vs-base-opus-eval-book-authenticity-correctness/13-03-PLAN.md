<objective>
Grade the book authenticity (fidelity to the book's mechanics) of every applied refactoring in the graded corpus, via the oracle agent (the only DST-04 channel to .oracle/). Normalize each run's own diff.patch + answer.md into a per-claim list in main context (never reading .oracle/), then package the claims to the oracle agent one book per call and record a pass/partial/fail verdict in the agent's own words.

Purpose: no per-run fidelity score of applied diffs against the books exists (Phase 10 graded the shipped catalog, not live run output). This produces the book-authenticity half of the head-to-head. Grading is oracle-agent calls on the Claude plan, NOT metered claude -p spend, so it runs after the wave-2 diffs exist without a new spend gate.
Output: grading/book-authenticity/claims.json (per-claim list) and grading/book-authenticity/fidelity.json (oracle verdicts + per-cell/arm tally + Pass@k), both tracked + ASCII-only + DST-04-clean.
</objective>
