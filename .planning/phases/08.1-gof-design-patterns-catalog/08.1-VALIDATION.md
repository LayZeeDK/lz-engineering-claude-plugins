- **After every task commit:** Run the single relevant checker (`node tools/check-gof.mjs` after a gof-leaf batch; `node tools/check-kerievsky.mjs` after linkify).
- **After every plan wave:** Run `cd .claude/skills/lz-refactor-workspace && npm run check && npm run typecheck` (all 8 checkers + tsc).
- **Before `/gsd:verify-work`:** Full suite GREEN + `claude plugin validate .`.
- **Max feedback latency:** ~30 seconds.
